& 安装启动

打开👉AList Release下载要部署的系统对应的文件（最新的版本前端已经与后端打包在一起，无需再次下载前端文件）。

打不开的可以在这里下载：

  • Linux系统：Linux一键部署请参考官方文档
  • Windows系统：解压下载的 zip 压缩包，启动 alist-windows-4.0-amd64.exe 即可。

& 设置

  • 当看到输出 start server @ 0.0.0.0:5244 且之后没有报错后，就表示运行成功了

  • 首次运行会输出初始密码：Your passwors：XXXXXX，记住这个密码

  • 程序默认监听5244端口，浏览器打开 http://你的ip:5244 就可以看见登录页面了，登录需输入密码，见上一步

  • 默认设置下IP是127.0.0.1

  • 点击右侧账号添加你想挂载的云盘账号

  • 不同云盘添加方式略有不同，具体请参考官方文档

  • 前端、后端中可以进行一些自定义设置

  • 这里以添加 阿里云盘 账号为例：

    • 点击添加，类型选 阿里云盘

    • 虚拟路径即你要挂载到的位置，如果你要挂载到根目录，即为 /，这里可以写 /阿里云盘

    • 索引 有多个账号时，用于排序，数字越小越靠前。比如我挂载了1个阿里和1个天翼，阿里索引填 0，天翼索引填 1，则阿里云盘排在前

    • down_proxy_url即下载代理地址，这里用不到，无视即可

    • 代理、webdav proxy及webdav direct，这里用不到，无视即可

    • 刷新令牌，用阿里云盘App扫描官方文档内给出的二维码获取

  • 根目录file_id，浏览器打开阿里云盘官网登录，点进去你要设置挂载的文件夹的url链接，如： https://www.aliyundrive.com/drive/folder/XXXXXXX ，folder/后面的XXXXXXX即为根目录file_id

  • 排序与排序方向随意

  • 最后设置如图，点击保存：

  • 浏览器打开 http://你的ip:5244 就可以看见你的阿里云盘了
  • 默认设置下IP是127.0.0.1

工具2：花生壳

通过花生壳软件将内网映射至外网，便于他人访问。

& 下载地址：

花生壳 - Windows版|Version8.5

❕ 注意：安装路径必须是英文

& 设置

Alist部署成功后，下载安装花生壳，其实就是设置个内部端口5244就可以了。

  • 下载安装登录后，点击自定义映射 右边的 ⊕，创建一条映射
  • 应用类型选HTTPS，内网端口填5244- 即Alist的本地端口
  • 外网域名即是你的云盘挂载网址，这是花生壳免费赠送的加密壳域名，详见关于调整【HTTP映射】及免费赠送加密壳域名的通知
  • 右边诊断信息显示连接成功说明外网映射创建成功，点击我的应用下方的链接访问吧

& 其他问题

  1. 需Alist程序与花生壳同时启动，类似于将本地电脑作为服务器，关机、断网、程序关闭时网址将不可访问。
  2. 花生壳个人免费版提供每月1G访问流量，需要大流量访问时可以升级正式版或购买流量包。
  3. 在网页的预览、播放、下载使用的是云盘的服务，不会占用本地流量。
  4. 云盘短时间内下载量过大有概率导致限速或封号，解决方案参考负载均衡

网站安装样式

  1. 点击网站下方的管理或打开http://127.0.0.1:5244/进入管理页面，点击左侧设置-前端
  2. 标题 、logo、 音乐封面 、图标颜色 等自由设置
  3. 自定义头部与自定义body处填入相应代码
